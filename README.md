# 🎬 Video Downloader Bot

**Telegram бот для скачивания видео с YouTube, TikTok, Instagram, Twitter/X и 1000+ других сайтов.**

---

## ✨ Возможности

- 📥 Скачивание видео в высоком качестве (с аудио)
- 🎵 Поддержка 1000+ сайтов (YouTube, TikTok, Instagram, Twitter, Vimeo, Facebook, Twitch, Reddit и др.)
- 📊 Информация о видео (название, автор, длительность, просмотры, лайки)
- 🔐 Обязательная подписка на каналы (настраивается админом)
- 👑 Админ-панель для управления обязательными каналами
- 📱 Отправка видео в виде видео-сообщения (можно смотреть сразу)
- 📖 Подробная справка по использованию

---

## 🚀 Быстрый старт

### Установка и запуск

```bash
# Клонируем репозиторий
git clone https://github.com/deffiofficalr-star/VIDEO-DOWNLOADER-BOT.git
cd VIDEO-DOWNLOADER-BOT

# Устанавливаем зависимости
pip install -r requirements.txt

# Запускаем бота
python bot.py
```

---

⚙️ Настройка

1. Получить токен бота

1. Напиши @BotFather в Telegram
2. Создай нового бота: /newbot
3. Скопируй токен

2. Вставить токен в код

Открой bot.py и замени:

```python
BOT_TOKEN = "ВАШ_ТОКЕН_ТЕЛЕГРАМ_БОТА"
```

3. Добавить админов

В bot.py укажи ID админов:

```python
ADMIN_IDS = [
    5413018519,  # твой ID
]
```

Узнать свой ID можно у бота @userinfobot

4. Добавить обязательные каналы (опционально)

```python
REQUIRED_CHANNELS = [
    "@bytefps",
]
```

❗ Бот должен быть администратором канала.
